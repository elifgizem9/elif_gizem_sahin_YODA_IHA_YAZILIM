## 1. Data Types 

Pythonda programlama yaparken kullandığımız verilerin türünü belirten kategorilerdir. Her veri tipi farklı türde bilgileri saklar ve bu bilgiler üzerinde farklı işlemler yapılmasına olanak tanır.

**Not:** Bir değişkenin veri tipini öğrenmek için `type()` fonksiyonunu kullanabilirsiniz.

### Temel Veri Tipleri

* **String :** Metinsel verileri ifade eder. Tek tırnak (`'...'`), çift tırnak (`"..."`) veya üç tırnak (`"""..."""`) arasına yazılır.
* **Integer :** Pozitif veya negatif tam sayıları ifade eder (örn: 10, -5, 0).
* **Float :** Ondalıklı sayıları ifade eder (örn: 3.14, -0.5).
* **Boolean :** Sadece iki değer alabilen veri tipidir: True veya False. Genellikle koşul kontrolünde kullanılır.

### Collection Veri Tipleri

* **List :** Farklı veri tiplerini bir arada tutabilen, sıralı ve değiştirilebilir bir koleksiyondur. Köşeli parantez [] ile tanımlanır.
* **Tuple :** Listelere benzer sıralıdır ancak değiştirilemez. Parantez () ile tanımlanır.
* **Dictionary :** key:value çiftlerinden oluşan, sırasız ve değiştirilebilir bir koleksiyondur. Süslü parantez {} ile tanımlanır.

### Veri Tipleri Örnekleri

# String
isim = "Ahmet"
mesaj = 'Python öğreniyorum.'

# Numeric
yas = 25 # Integer
pi_sayisi = 3.14 # Float

# Boolean
a= True

# List
meyveler = ["elma", "muz", "çilek"]
meyveler.append("portakal") # Listeler değiştirilebilir

# Dictionary

    "ad": "Zeynep",
    "yas": 30,
    "meslek": "Mühendis"
}

kisi["yas"] = 31 # Sözlükler değiştirilebilir

print(type(yas)) # Çıktı: <class 'int'>
print(type(kisi)) # Çıktı: <class 'dict'>

## 3. Loops 
​Bir kod bloğunu belirli bir koşul sağlandığı sürece veya bir koleksiyon üzerindeki her bir eleman için tekrar tekrar çalıştırmamızı sağlayan yapılardır.
#​for Döngüsü
​Genellikle bir koleksiyonun elemanları üzerinde gezinmek veya belirli bir sayıda tekrar yapmak için kullanılır. range() fonksiyonu ile sıkça kullanılır.

# Liste üzerinde gezinme
ogrenciler = ["Ali", "Veli", "Ayşe"]

for ogrenci in ogrenciler:
    print(f"Hoş geldin {ogrenci}") # burada f koyarak süslü parentez içindeki kodu çalıştırıp yerine yazabiliyoruz.

# String üzerinde gezinme
for harf in "Python":
    print(harf)

# range() ile belirli sayıda tekrar
# 0'dan 4'e kadar sayıları yazdırır
for i in range(5):
    print(i) # 0, 1, 2, 3, 4

# 1'den 10'a kadar (11 dahil değil) 2'şer atlayarak
for i in range(1, 11, 2):
    print(i) # 1, 3, 5, 7, 9

## while Döngüsü
​Belirli bir koşul True olduğu sürece çalışmaya devam eder. Koşul False olduğunda döngü sona erer.
while döngülerinde koşulun bir noktada False olmasını sağlamak gerekir aksi takdirde infinite loop oluşur.

# 1'den 5'e kadar sayıları yazdırma
sayac = 1
while sayac <= 5:
    print(sayac)
    sayac = sayac + 1 # Koşulu değiştirir

# Döngü Kontrol İfadeleri: break ve continue
sayac = 0
while True: # Potansiyel sonsuz döngü
    sayac += 1
    if sayac % 2 == 0:
        continue # Döngünün bu adımını atla, başa dön 
    print(sayac)
    
    if sayac >= 9:
        break # Döngüyü tamamen kır dışarı çık

## 4. Functions 
​Belirli bir görevi yerine getirmek için yazılmış tekrar tekrar kullanılabilen kod bloklarıdır. Kodun daha modüler organize ve okunabilir olmasını sağlarlar.
​def anahtar kelimesi ile tanımlanırlar.
Parameter: Fonksiyonun aldığı değişkenlerdir (örn: def topla(a, b)).
Argument: Fonksiyonu çağırırken bu parametrelere verdiğimiz gerçek değerlerdir (örn: topla(5, 10)).
​return anahtar kelimesi ile fonksiyon bir değer döndürebilir. Eğer return kullanılmazsa, fonksiyon varsayılan olarak None döndürür.

### ​Fonksiyon Tanımlama ve Kullanma Örnekleri

## 5. Class 
​Sınıflar, Nesne Yönelimli Programlamanın Object-Oriented Programming - OOP temel taşıdır. Sınıflar kendi veri tiplerimizi oluşturmak için kullandığımız şablonlar veya kalıplardır.
Class: Bir nesnenin özelliklerini attributes ve davranışlarını methods tanımlayan şablondur. (Örn: Araba sınıfı).
Object / Instance: Bir sınıftan şablondan üretilmiş somut bir örnektir. (Örn: benimarabam veya kirmiziferrari).
​Nitelik Attribute: Bir nesnenin sahip olduğu özelliklerdir. (Örn: Araba sınıfı için renk, model, hiz).
Method: Bir nesnenin yapabileceği davranışlardır sınıf içinde tanımlanan fonksiyonlardır. (Örn: Araba sınıfı için calistir(), hizlan(), dur()).

# __init__ Metodu Constructor
​Bir sınıftan nesne oluşturulduğu anda otomatik olarak çalışan özel bir metottur. Genellikle nesnenin başlangıç niteliklerini belirlemek için kullanılır. self parametresi, o anda oluşturulan nesnenin kendisini temsil eder.

### ​Sınıf Tanımlama ve Nesne Oluşturma Örnekleri

# 1. Sınıf Tanımlaması Şablon
class Araba:
    # Constructor metodu
    def __init__(self, marka, model, yil, renk):
        # Bunlar niteliktir attribute
        self.marka = marka
        self.model = model
        self.yil = yil
        self.renk = renk
        self.calisiyormu = False # Varsayılan değer
        self.hiz = 0

    # Bunlar metot
    def calistir(self):
        if not self.calisiyormu:
            self.calisiyormu = True
            print(f"{self.marka} {self.model} çalıştırıldı.")
        else:
            print("Araç zaten çalışıyor.")

    def durdur(self):
        if self.calisiyormu:
            self.calisiyormu = False
            self.hiz = 0
            print(f"{self.marka} {self.model} durduruldu.")
        else:
            print("Araç zaten durmuş durumda.")
    
    def hizlan(self, miktar):
        if self.calisiyormu:
            self.hiz += miktar
            print(f"Hız artırıldı. Yeni hız: {self.hiz} km/s")
        else:
            print("Araç çalışmıyor hızlanamaz.")

    def bilgilerigoster(self):
        print(f"Araç Bilgileri")
        print(f"Marka: {self.marka}")
        print(f"Model: {self.model}")
        print(f"Yıl: {self.yil}")
        print(f"Renk: {self.renk}")
        print(f"Durum: {'Çalışıyor' if self.calisiyormu else 'DURUYOR'}")
        print(f"Hız: {self.hiz} km/s")

# 2. Nesnelerin Oluşturulması Instance yaratma
# Araba sınıfı şablonunu kullanarak iki farklı nesne oluşturuyoruz.
araba1 = Araba("Toyota", "Corolla", 2022, "Beyaz")
araba2 = Araba("Ford", "Mustang", 1969, "Kırmızı")
